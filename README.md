## 安装

```bash
pnpm add @zesenzhu/i18n-scripts -D
```

## 使用

在项目 `package.json` 的 `scripts` 增加脚本命令

### i18nJson2xls

i18n 国际化文案 json 文件统一整合转换为 excel 文件

```bash
i18n-scripts i18nJson2xls
```

### xls2i18nJson

国际化 excel 文件转换分解为多个 json 文件

```bash
i18n-scripts xls2i18nJson
```
